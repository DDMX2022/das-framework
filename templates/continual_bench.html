What is catastrophic forgetting?

Standard neural networks have a single set of shared weights. When you train on a new task, gradient updates shift those weights toward the new task's optimum — erasing what was learned before. This is catastrophic forgetting. On Split-MNIST, a fine-tuned MLP's accuracy on earlier tasks often drops to near 50% (random binary chance) by the time it finishes task 4.

DAS's structural solution: each task gets a dedicated frozen leaf. Once trained and locked, its weights can never be touched by subsequent training. The proof is cryptographic — SHA-256 hash verification.

BWT (Backward Transfer) measures average accuracy drop on previous tasks after all training is done. DAS targets BWT ≈ 0.00. Fine-tuned MLP typically shows BWT < −0.20.
