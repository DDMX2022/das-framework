What you're watching

This demo walks through the full DAS lifecycle on synthetic data — three distinct domains, each with its own decision boundary. The tree metaphor is literal: roots anchor the routing system, the trunk is the shared Stem Router, each branch carries data to a specialist leaf, and the canopy emerges when all leaves are active.

Phase 1
Router trains
Learns to sort 600 samples from 2 domains into the right branch using a linear gate + softmax.
Phases 2–3
Leaves train in isolation
Leaf 0 and Leaf 1 each train on their own domain only. The other leaf's weights are frozen and untouched.
Phase 4 ✦
Grafting
A third domain arrives. A new leaf sprouts — existing leaves stay frozen. This is modular growth.
Proof
SHA-256 fingerprints
Leaf weights are hashed before and after grafting. Byte-identical = zero catastrophic forgetting.

▸ What the benchmark measures

Dataset: sklearn's handwritten digits — 1,797 real digit images (8×8 px, 64 features, 10 classes). Split into 3 domains by digit group, each with a distinct binary classification task. The router is trained on all 3 domains together, then each leaf trains in isolation on only its own domain.

▸ DAS Forest vs. Baseline MLP — how they differ
DAS Forest
Router hard-routes each sample to exactly one leaf. Each leaf trains only on its domain — it never sees data from other domains. Leaves are frozen after training. A new domain can be added by grafting a new leaf without touching existing ones.
Baseline MLP
A single network sees all domains at once. It must learn all three binary tasks simultaneously using the same shared weights. Learning one task can interfere with another — this is catastrophic forgetting.
Reading the results table: Domain test accuracy tells you how well each approach solved its specific binary task on held-out data. Same parameter budget — any accuracy gap is structural, not a size advantage. The isolation proof SHA-256 hashes leaf weights before and after grafting — UNCHANGED means zero bytes moved in existing leaves while the new one was added.
