What this stress test does

Dataset: MNIST — 70,000 handwritten digit images (60k train / 10k test), each flattened to 784 pixel features. This is the canonical deep-learning benchmark.

Architecture: A 10-leaf DAS Forest. Each leaf is a deep specialist MLP [784 → 256 → 64 → 2]. They are trained one at a time, in complete isolation — no shared weights, no joint loss, no cross-contamination between leaves.

Phase 1 — RouterThe Stem Router learns 10-way routing on raw pixels. It decides which leaf should handle each incoming digit image.
Phases 2–11 — LeavesEach leaf trains on a 1-vs-rest binary task: "is this digit N, or not?" Balanced dataset: equal positive and negative examples. All other leaves are frozen during this phase.
Phase 12 — BaselineA single monolithic MLP [784 → 512 → 256 → 2] trains on all 10 classes simultaneously, the standard approach. Used as the comparison point.

How to read the results

DAS leaf accEach leaf's accuracy on its own 1-vs-rest binary test set. High accuracy = the leaf correctly identifies its digit vs. all others.
Baseline accThe monolithic MLP's accuracy across all 10 classes. A different (harder) task — included so you can see both approaches reach ~98%, but DAS gets there with fully isolated specialists.
Hash unchanged?After each leaf is frozen, its full weight tensor is SHA-256 hashed. Every subsequent training round (other leaves) is verified against that hash. UNCHANGED ✓ means zero bytes changed — cryptographic proof of isolation.
Forgetting proof45 pairwise checks total (leaf 0 verified during leaves 1–9, leaf 1 during 2–9, etc.). All must pass. This is the core claim of DAS: you can add specialists without disturbing what already exists.
