What this benchmark tests

Each dataset has a natural domain structure — groups of samples that are statistically distinct. DAS trains one specialist leaf per domain in isolation. The baseline is a single MLP trained on all domains mixed together.

DAS leaf Sees only its domain. Trains to convergence without interference from other domains.
Baseline MLP Sees all domains at once. Must learn a shared representation — prone to cross-domain interference.
Forgetting proof SHA-256 hash of each leaf's weights before and after other leaves train. PASS = zero bytes changed.
Fair comparison Baseline is sized to have roughly the same total parameter count as the DAS forest.
